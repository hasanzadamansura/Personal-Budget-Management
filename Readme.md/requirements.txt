# 💰 Sadə Büdcə İdarəetmə Tətbiqi

Universtet layihəsi — Python (Flask) backend + HTML/CSS/JS frontend.

---

## 📁 Qovluq Strukturu

```
budget-app/
├── backend/
│   ├── app.py              ← Flask API serveri
│   ├── database.py         ← SQLite əməliyyatları
│   ├── models.py           ← Məlumat modelləri
│   └── requirements.txt    ← Python asılılıqları
├── frontend/
│   ├── index.html          ← Ana səhifə (Login + Tətbiq)
│   ├── style.css           ← Dizayn
│   └── script.js           ← API ilə əlaqə, UI məntiqi
├── database/
│   └── budget.db           ← SQLite DB (avtomatik yaranır)
└── README.md
```

---

## ⚙️ Quraşdırma və İşə Salma

### 1. Python asılılıqlarını qur

```bash
cd backend
pip install -r requirements.txt
```

### 2. Backend-i işə sal

```bash
python app.py
```

Server `http://localhost:5000` adresində işə düşür.

### 3. Frontend-i aç

`frontend/index.html` faylını brauzerdə aç  
(iki dəfə klikləmək kifayətdir).

---

## 🔗 API Endpoint-lər

| Method | URL | Təsvir |
|--------|-----|--------|
| POST | `/api/register` | Qeydiyyat |
| POST | `/api/login` | Giriş |
| POST | `/api/logout` | Çıxış |
| GET  | `/api/me` | Cari istifadəçi |
| GET  | `/api/balance` | Balans məlumatı |
| POST | `/api/income` | Gəlir əlavə et |
| POST | `/api/expense` | Xərc əlavə et |
| GET  | `/api/transactions` | Bütün əməliyyatlar |
| DELETE | `/api/transaction/<id>` | Əməliyyatı sil |
| GET  | `/api/report/monthly` | Aylıq hesabat |

---

## 🛠️ Texnologiyalar

- **Backend:** Python 3, Flask, SQLite3, Werkzeug
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **DB:** SQLite (fayl əsaslı, quraşdırma tələb etmir)

---

## 👤 İstifadə

1. Tətbiqi açın → **Qeydiyyat** tabına keçin
2. İstifadəçi adı və şifrə ilə qeydiyyatdan keçin
3. **Giriş** edin
4. Gəlir/Xərc əlavə edin, balansınıza baxın
5. **Aylıq Hesabat** tabından aylıq statistikanı izləyin
